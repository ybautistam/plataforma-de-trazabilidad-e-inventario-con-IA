# plataforma-de-trazabilidad-e-inventario-con-IA
---

## Objetivo 

- Control eficiente del inventario.
- Trazabilidad completa de productos.
- IAnálisis inteligente de datos.
- Soporte para toma de decisiones basada en IA.


## Stack Propuesto para el desarrollo 

- **Backend:** ASP.NET Core (.NET)
- **Frontend:** React
- **Base de Datos:** PostgreSQL
- **Control de Versiones:** Git
- **Metodología:** Scrum

---

#SDLC Elegido

Se utilizará **Scrum ** como metodología de desarrollo.

## Justificación

- Desarrollo incremental e iterativo.
- Integración progresiva de módulos:
  - Gestión de Inventario
  - Trazabilidad de productos
  - Reportes
  - Inteligencia Artificial (predicción y alertas)
- Adaptabilidad a cambios.
- Trabajo paralelo entre frontend y backend.

## Organización del Trabajo

- Sprints de 2 semanas.
- Daily meetings.
- Sprint Planning.
- Sprint Review.
- Product Backlog priorizado (Lista y ordenamiento de todas las tareas del proyecto) .
